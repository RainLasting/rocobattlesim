## 技术实现要点

### 1. 新伤害计算公式

```python
damage = math.ceil(
    (attacker_atk / actual_defender_def) * 0.9 *
    (skill_power * response_multiplier + power_bonus) *
    ability_level *
    (1 + power_boost) *
    weakness_multiplier *
    (1 + weather_effect) *
    (1 - damage_reduction)
)
```

### 2. 能力等级计算公式

```python
def calculate_ability_level(attack_boost, defense_reduction, attack_reduction, defense_boost):
    return (1 + attack_boost + defense_reduction) / (1 + attack_reduction + defense_boost)
```

### 3. API参数更新

伤害计算API新增了以下参数：
- `attack_boost`: 我方攻击提升百分比
- `defense_reduction`: 敌方防御降低百分比
- `attack_reduction`: 我方攻击降低百分比
- `defense_boost`: 敌方防御提升百分比
