# 对战模拟器开发计划

## 1. 项目概述

本项目旨在开发一个基于Python Flask和Tailwind CSS的对战模拟器，用于测试基于当前属性、技能和宠物设定的实际对战情况。模拟器采用前后端分离的架构，提供直观的用户界面和准确的战斗计算。

## 2. 技术栈

- **后端**: Python Flask
- **前端**: HTML/CSS/JS + Tailwind CSS
- **数据存储**: JSON文件
- **部署方式**: 本地运行

## 4. 核心功能模块

### 4.1 宠物选择系统

- 实现双方宠物选择功能
- 从现有的JSON文件加载宠物数据
- 显示已选宠物的基础属性信息（属性固定）
- 提供宠物列表供用户选择

### 4.2 技能配置模块

- 从宠物可装备技能库中选择4个技能
- 显示所选技能的详细参数和效果说明
- 技能数据从现有的JSON文件加载

### 4.3 修正系统

- 实现增益修正选择功能，对指定属性提供20%加成
- 实现减益修正选择功能，对指定属性提供10%削弱
- 提供三项个体值配置，每项个体值上限为10

### 4.4 属性计算系统

- 生命计算公式：⌈(1.7×hp+个体值×0.85×6+70)×(1+修正)+100⌉（向上取整）
- 其他属性计算公式：⌈(1.1×属性种族值+个体值×0.55×6+10)×(1+修正)+50⌉（向上取整）
- 实时计算并显示计算结果

### 4.5 战斗计算系统

- 伤害计算公式：⌈(进攻方攻击/防御方攻击)×0.9×(技能威力×应对倍率+威力加成)×能力等级⌉（向上取整）
- 能力等级计算公式：(1+我方攻击提升+敌方防御降低)/(1+我方攻击降低+敌方防御提升)
- 技能威力计算公式：威力提升数值×克制关系×(1-减伤)×天气影响
- 提供应对倍率选择（默认为1）
- 提供威力加成输入框（默认为0）
- 提供能力等级相关参数配置
- 提供威力提升数值输入框（默认为0）
- 提供减伤输入框（默认0）
- 提供天气影响输入框（默认0）
- 计算并显示几次攻击可以确认击杀

### 4.6 系别克制系统

- 单系克制关系为2倍
- 双系都被克制关系为3倍
- 抵抗关系为克制的倒数
- 具体系别和计算逻辑参照文件 `attr/attr_map.json` 和 `view/attrexample.py`

## 5. 前端界面设计

- 采用左右分栏布局
- 左侧：宠物选择和技能配置
- 右侧：战斗计算和结果显示
- 使用Tailwind CSS实现响应式设计
- 提供直观的用户交互界面

## 6. 后端API设计

- `/api/pets`: 获取宠物列表
- `/api/pet/{pet_id}`: 获取宠物详细信息
- `/api/skills/{pet_id}`: 获取宠物可装备技能
- `/api/skill/{skill_id}`: 获取技能详细信息
- `/api/calculate/attributes`: 计算宠物属性
- `/api/calculate/damage`: 计算伤害
- `/api/calculate/weakness`: 计算属性克制关系
